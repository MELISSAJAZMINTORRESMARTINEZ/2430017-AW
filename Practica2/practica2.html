Introduccion y contexto

¿Que es JavaScript y sus prinicipales usos en la web?

JavaScript es un lenguaje de programacion que los desarrolladores utilizar para hacaer pagias web interactivas. Desde utilizar fuentes de redes sociales a mostrar animaciones y mapas interativos, las funciones de JS mejorar la experienca del usuario al momento estar en el sitio web. Como lenguaje de scripting del lado del servidor, se trata de una de las principaes tecnologias de la World Wide Web. Por ejemplo: Al momento en que navegas por internet o pagina web puedes encontrar un carrusel de imagenes, un menu desplegable o quiza cambien de manera dinamica los elementos de color en una pagina Web

JavaScript fue creado por Brendan Eich en mayo de 1995, mientras este trabajaba para Netscape Communicactions Este se lanzo por primera vez con Netscape 2 a principios de 1996. Originalmente se se llamaria LiveScript pero se cambio el nombre por una decision de marketing que intento capitalizar la popularidad del lenguaje Java de Sun Microsystem.

El proposito con el que se creo fue para permitir que las paginas fueran mas interactivas, y no solamente fueran paginas estetcias, tambien se diseño para generar contenido de forma dinamica, sin la necesidad de recargar la pagina completa. La idea era tener un leguaje facil y sencillo de usar para que cualquiera pudiera darle un poco mas de vida a las paginas web, agregando funciones directamente en el navegador y haciendo la experiencia del usuario mas dinamica del lado del cliente.

Por que JavaScript y Java no son lo mismo?

Java es un leguanje de programacion orientado a objetos el cual se usa para poder crear apliciaciones grandes, seguras y que sean funcionables en diferentes sistemas. En cambio JavaScript es un leguanje mas ligero para poder darle vida a las paginas web dentro del navegador, el cual tiene una forma de escribir mas flexible y que es menos estricta

Los nombres no estan directamente enlazados o relacionados; el parecido fue mas que nada por marketing en los 90 para poder aprovechar las relevancia y la fama que tenia Java en ese entonces.

¿Como se trabajan juntos JavaScript, HTML y CSS

JavaScript, HTML y CSS se c¿juntan para darle forma a una pagian: HTML arma la estrcutrua y el contendio,CSS se encarga de que todo se vea bien ya sea con colores, estilos y diseño, y por ultimo JavaScript le da vida a una pagina con interactividad, haciendo que los elementos reaccionen a lo que el usuario hace.

Fundamentos del lenguaje

Sintaxis basica en JavaScript

JavaScript es un lenguaje de programacion que se caracteriza por tener una sintaxis similar a la del lenguaje C, lo que lo hace facil de aprender para aquellos que ya estan familiarizados con este lenguaje. La sintaxis de JavaScript es bastante flexible y permite a los desarrolladores escribir codigo de manera concisa y legible. A continuacion, se presentan algunos conceptos basicos de la sintaxis de JavaScript:

     let igual = 5 == "5";  (Comparación: ==, ===, !=, !==, >, <, >=, <=) 
     let logico = true && false; (Lógicos: &&, ||, !)
  • Condicionales: Se usa para controlar el flujo de ejecucion del codigo, asi mimso permitiendo que se ejecuten diferentes blqoues de instruccion basada en si una condicion es verdadera o falsa, alguas de estas son if, if else, else if, etc. Por ejemplo: 
    if (edad >= 18) {

Diferencia entre var, let y const

La principal difrenecta entre var, let, y const tiene que ver con donde se puede utilizar la variable y si su valor se puede cambiar o no. Var tiene alcance de funcion, se puede reasignar y re-declarar, pero hoy en dia se evita porque puede llegar a generar errores. Let tiene alcance de bloque, se puede cambiar su valor pero no se puede declarar dos veces en el mismo bloque, lo que lo hace mas seguro. Const, tambien con alcance de bloque, no se puede reasignar ni re-declarar y debe inicializarse al declarala, por eso se ua para valores que no cambian

Asi que, usa const para cosas que siempre van a ser iguales let para valores que cambian, por ultimo var aunque puede que sea confunso igual se puede seguir usando.

Su diferencia principal es que los arrays almacenan colecciones ordenadas de valor utlizando inidces numerico (como [0], [1]) , mientars que los objetos almacenan colecciones de pares clave-valor, donde cada valor se accede mediante una clave unica (como nombre: "Ana", edad: 20).

Operadores aritmeticos, logicos y relacionales

En JavaScript, los operadores son simbolos que le dicen al programa que hacer con los valores. Hay tres principales que se utilizan mucho en la programacion: aritmeticos, logicos y relacionales.

Operadores Aritmeticos: Estos operadores se utilizan para realizar operaciones matematicas basicas como suma, resta, multiplicacion y division. Algunos ejemplos son:

  • + (suma)
  • - (resta)
  • * (multiplicacion)
  • / (division)
  • % (modulo, que da el residuo de una division)
Operadores Logicos: Estos operadores se utilizan para combinar o invertir valores booleanos (verdadero o falso). Algunos ejemplos son:
  • && (AND logico, devuelve verdadero si ambos operandos son verdaderos)
  • || (OR logico, devuelve verdadero si al menos uno de los operandos es verdadero)
  • ! (NOT logico, invierte el valor del operando)
Operadores Relacionales: Estos operadores se utilizan para comparar dos valores y devolver un valor booleano. Algunos ejemplos son:
  • == (igualdad, devuelve verdadero si los operandos son iguales)
  • === (igualdad estricta, devuelve verdadero si los operandos son iguales y del mismo tipo)
  • != (desigualdad, devuelve verdadero si los operandos no son iguales)
  • !== (desigualdad estricta, devuelve verdadero si los operandos no son iguales o no son del mismo tipo)
  • > (mayor que)
  • < (menor que)
  • >= (mayor o igual que)
  • <= (menor o igual que)
Estos operadores son fundamentales para escribir codigo en JavaScript y se utilizan en una amplia variedad de situaciones, desde realizar calculos matematicos hasta tomar decisiones basadas en condiciones.

Control de flujo

¿Como funcionan los Condicionales if, else y switch?

  • if: se usa para ejecutar un bloque de codigo solo si es que se cumple una condicion. Ejemplo 
    let edad = 18;
    if (edad >= 18) {
    console.log("Eres mayor de edad");
    }
  • else: se usa junto con if para ejecutar un bloque de codigo alternativo si la condicion del if no se cumple. Ejemplo 
    let edad = 16;
    if (edad >= 18) {
    console.log("Eres mayor de edad");
    } else {
    console.log("Eres menor de edad");
    }
  • switch: sirve cuando queremos comparar una misma variable con multiples valores posibles de manera un poquito mas ordenada que con muchos if. Ejemplo
    • let dia = 3;
    switch (dia) {
    case 1:
    console.log("Lunes");
    break;
    case 2:
    console.log("Martes");
    break;
    case 3:
    console.log("Miércoles");
    break;
    default:
    console.log("Día no válido");
    }

Diferencia entre for, while y do while

  • for: Este se utiliza cuando sabes cuantas veces vas a querer repetir algo. Ejemplo 
    for (let i = 0;  i < 5; i++){
    console.log(i);
    }
  • while: Este se utiliza cuando no sabes cuantas veces vas a querer repetir algo, pero si sabes la condicion para que se siga repitiendo. Ejemplo 
    let i = 0; 
    while (i < 5) { 
    console.log(i); 
    i++; 
    }
  • do while: Este es similar al while, pero la diferencia es que el bloque de codigo se ejecuta al menos una vez antes de verificar la condicion. Ejemplo: 
    let i = 0; 
    do { 
    console.log(i); 
    i++; 
    } while (i < 5);

Para que sirven las palabras break y continue?

En JavaScript, break sale por completo del un bulce, transfiriendo el control del programa a la siguiente instruccion del bulce. Luego tenemos continue termina la iteracion actual del bucle y pasa rapidamente a la siguiente, sin salir del bulce

Como se declara una funcion y como se llama?

En JavaScript, declarar una funcion significa crear un bloque de codgio que puedes usar varias veces, y llamarla es ejecutarla cuando la necesites 

function saludar(nombre) { 
console.log("Hola, " + nombre + "!"); 
} 

// Llamar a la función 
saludar("Ana"); // Salida: Hola, Ana!

Funcion con parametros y valor de retorno

Al momento que una funcion tiene parametros y un valor de retorno significa lo siguiente:

  • Parametros: son los datos que se les da a la funcion para que trabaje con ellos. Son como tipo entradas que la funcion recibe. 
    function sumar(a, b){
    return a + b}
  • Valor de retorno: es el resultados que la funcion va a devolver despues de hacer todo. Se usa la palabra return. 
    let resultado = sumar (2, 3) //5, el valor que devuelve la funcion 
    console.log(resultado);

Que son las funciones flecha y que ventajas tiene?

Las funciones flecha son una forma mas corta de escribir funciones. Ejemplo 

//Funcion normal 
function sumar(a, b) { 
return a + b; 
} 

//Funcion flecha 
const sumar = (a, b) => a + b;
Las ventajas de las funciones flecha son:
  • Sintaxis mas corta: Son mas concisas y faciles de leer.
  • No tienen su propio this: Heredan el valor de this del contexto donde fueron creadas, lo que puede evitar confusiones en ciertos casos.
  • No se pueden usar como constructores: No se pueden usar con la palabra clave new, lo que puede ayudar a evitar errores accidentales.

Scope y como afecta a las variables?

El Scope define donde se puede usar una variable dentro del codigo. Determianr la "zona" en la que una variable existe y puede ser accedida, en si afecta a las variables porque define donde se pueden usar y ayuda a evitar conflictos o errores en el codigo.

Objetos y funciones

Como declarar un objeto y acceder a sus propiedades?

En JavaScript, para declarar un objeto solo necesitas usar llavas {} y dentro poner pares clave-valor, donde cada clave es el nombre de la propiedad y el valor es lo que guarda esa propiedad. Por ejemplo: 

let persona = { 
nombre: "Ana", 
edad: 25, 
trabajo: "Cuidando gatitos" 
};
Para acceder a las propiedades de un objeto, puedes usar la notacion de punto (.) o la notacion de corchetes ([]). Por ejemplo: 
// Notación de punto 
console.log(persona.nombre); 
// Notación de corchetes 
console.log(persona["edad"]);

Diferencia entre propiedad y un metodo objeto?

En JavaScript, una propiedad es un dato que describe alguna caracteristica del objeto, como un nombre, edad o color. En cambio, un metodo es una funcion que forma parte del objeto y srive para realizar alguna accion o comportamiento usando sus propiedades

Metodos basicos (push, pop, map)

  • Push: agrega uno o mas elementos al final del array. Ejemplo 
    let frutas = ["manzana", "platano"];
    frutas.push ("fresa");
    console.log (frutas) //manzana, platano, fresa
  • Pop: elimina el ultimo elemento del array. Ejemplo 
    frutas.pop() 
    console.log (frutas); //manzana, platano
  • Map: Crea un nuevo array aplicando una funcion a cada ultimo del original 
    let numeros = [1, 2, 3]; 
    llet dobles = numeros.map(n => * 2); 
    console.log(dobles); //[2, 4, 6]

Otros temas basicos

Que es el DOM y por que es importante?

El Modelo de Objeto de Documento (DOM) es una representacion estructurada en forma de arbol de una pagina web que actuca como un puente entre el codigo JavaScript y el contenido HTML, permitiendo manipular dinamicamente su estructura y su estilo.

Es importante porque permite que JavaScript haga que las paginas web sean dinamicas e interactivas. Con el, se pueden respodern a eventos del usuario , cambiar el contenido de los elementos, modificar su estilo y actualizar la info sin recargar la pagina. Ademas, su estrandarizacion asegura que estas acciones funcionen de manera consistente en distintos navegadores.

Eventos en JavaScript

Es cualquier accion que ocurre en la pagina y a la que el programa puede reaccioanr como algo que hacer el usuario o que sucede automaticamente.

  • Click: cuando el usuario hace clic en un boton o enlace
  • Mouseover: cuando el usuario mueve el cursor sobre un elemento
  • Keydown: cuando el usuario presiona una tecla
Estos eventos de aqui permiten que las paginas puedan responder y sean mucho mas interactivas.

Diferencia entre alert, prompt y console.log

En JavaScript, alert, prompt, y console.log se usan para mostrar informacion, pero estas funcionan de manera distina.

  • Alert: muestra un mensaje en una ventana emergente que el usuario debe cerrar.
  • Prompt: muestra un mensaje en una ventana emergente y perimite que el usuario ingres un valor.
  • Console.log: muestra informacion en la cosola del navegador, util para depuracion, pero no interrumpe la paginani pide interaccion al usuario.
Alert es solo para avisos, prompt para pedir datitos y console.log para ver infromacion mientras se va desarrollando el codigo.

Abrir consola del navegador y ventajas de usarla

  • Presionando F12 o Ctrl+Shift+I (Cmd+Option+I en Mac) y luego seleccionando la pestaña Console.
  • Haciendo clic derecho en la pagina, eligiendo Insepeccionar y luego yendo a Console
Las ventajas de usar la consola del navegador son:
  • Perimite ver los errores y advertencias que ocurren al ejecutar JavaScript
  • Facilita probar fragmentos de codgio sin modificar los archivos originales
  • Por ultimo ayuda a insepccionar valores de variables y objetos en tiempo real para entender como funciona el programa.
